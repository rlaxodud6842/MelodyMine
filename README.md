# MelodyMine

**MelodyMine**는 특정 채널을 주목하고 있다가, 새로운 영상이 올라올 경우 동영상, 음원으로 저장해주는 프로그램입니다.

## 설치방법
+ 파이썬 3.7이상이 요구됩니다.
+ 가상환경 설정
     `python3 -m venv venv` or `python -m venv venv`
+ 가상환경 활성화
   Windows: `./venv/Scripts/activate` or `cd ./venv/Scripts` AND `activate`
+ requirements 설치
    * `pip install -r requirements.txt`
 
## 실행방법
+ `python ./main.py`

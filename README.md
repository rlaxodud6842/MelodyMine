# MelodyMine
Youtube에 특정 채널에 올라오는 영상을 자동으로 mp3파일로 저장해주는 프로그램.

## 프로젝트 목표
Youtube 특정 채널을 계속 주시하면서 새로운 영상이 올라오면 mp3로 저장해주는것.

### 기대효과 
> 번거롭게 새로운 영상이 올라왔나 확인할 필요 줄고, 영상을 mp3로 바꾸는 번거로움도 줆.
